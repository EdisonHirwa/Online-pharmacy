# Hospital Management System

## Overview
A comprehensive Hospital Management System built with Pure PHP (Backend) and React.js (Frontend).

## Tech Stack
- **Backend**: Pure PHP, MySQLi, MVC Pattern.
- **Frontend**: React.js, Material UI, Recharts, Vite.
- **Database**: MySQL.

## Installation
1. Import `database.sql` into your MySQL server.
2. Configure `Backend/config/connection.php` if needed.
3. Run `npm install` and `npm run dev` in the `frontend` directory.
4. Access the app at `http://localhost:5173`.

## Folder Structure
- `Backend/`: PHP API and logic.
- `frontend/`: React application.
- `database.sql`: Database schema.
